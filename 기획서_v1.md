# 미용실 잔액 관리 앱 기획서 v1

## 핵심 컨셉

> **"앱 설치 없이, 핸드폰 뒷자리만 누르면 내 잔액 확인"**

---

## 문제 정의

1. 고객: 매번 "잔액 얼마예요?" 물어봐야 함
2. 미용실: 수기로 잔액 관리 → 실수 가능성
3. 새 앱 설치 거부감 → **웹 기반 필수**

---

## 핵심 기능

### 1. 초간단 로그인 (3가지 옵션)

| 방식 | 설명 | 난이도 |
|------|------|--------|
| **핸드폰 뒷자리 4자리** | 가장 간편, 설치 불필요 | ⭐ 쉬움 |
| 전화번호 인증 (SMS) | 보안 강화 | 중간 |
| Google 로그인 | 기존 계정 활용 | 중간 |

**MVP는 "뒷자리 4자리"로 시작!**

### 3. 이벤트/쿠폰 시스템

| 이벤트 종류 | 설명 |
|------------|------|
| 🎂 생일 쿠폰 | 생일 달에 자동 발급 |
| 🎁 첫 충전 보너스 | 10만원 충전 시 1만원 추가 |
| 📅 휴면 고객 쿠폰 | 3개월 미방문 시 할인 |
| 💳 대량 충전 혜택 | 30만원 이상 충전 시 추가 적립 |

### 4. 미용실 관리자용

- 고객 등록 (이름 + 전화번호)
- 잔액 충전/차감
- 서비스 가격 설정 (변경 가능)
- 쿠폰/이벤트 생성
- 고객 목록 및 통계

## 기술 스택 (제안)

| 구분 | 기술 | 이유 |
|------|------|------|
| 프론트엔드 | HTML/CSS/JS (단일 페이지) | 빠른 개발, 설치 불필요 |
| 백엔드 | Firebase | 무료 시작, 실시간 DB |
| 인증 | 전화번호 뒷자리 | 최소 마찰 |
| 호스팅 | Cloudflare Pages | 무료, 빠름 |

## 사용자 플로우

### 고객 플로우
```
QR코드 스캔 또는 링크 클릭
        ↓
  전화번호 뒷자리 입력
        ↓
    잔액 & 쿠폰 확인
```

### 미용실 플로우
```
관리자 로그인 (비밀번호)
        ↓
   고객 검색 (전화번호)
        ↓
  충전 또는 서비스 차감
        ↓
   자동으로 고객에게 반영
```
