### 2. 미용실 잔액 관리 앱
- 현금 결제 시 20% 할인

### 3. 연차 계산기

## 추천 순위

### 1위: 미용실 잔액 앱 (선정)

**선정 이유:**
- 본인이 직접 겪는 불편함 = 진짜 문제
- 미용실 사장님 + 고객 양쪽 모두에게 가치 제공
- 경쟁자가 거의 없음 (틈새시장)
- B2B SaaS = 월정액 수익 가능
- 다른 업종으로 확장 가능 (네일샵, 마사지샵 등)

### 2위: 연차 계산기

**이유:**
- 빠르게 만들 수 있음 (2-3일)
- 포트폴리오로 좋음
- 하지만 대부분 회사에 HR 시스템 있어서 시장 제한적

### 3위: 닮은 연예인

**이유:**
- 바이럴 가능성은 높지만
- AI API 비용 높음 (얼굴 인식, 이미지 분석)
- 이미 많은 앱 존재

## 결론

**"미용실 잔액 앱"을 1순위로 개발 진행**

- 본인이 실제 고객으로서 문제를 아는 게 가장 강력한 무기
- 다니는 미용실에서 바로 테스트 가능
- MVP 빠르게 만들고 → 실제 사용 → 피드백 → 개선
